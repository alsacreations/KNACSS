Styles

Fichiers de styles chargés dans l'ordre des layers CSS :

  1. Layer reset :
    • bretzel-reset.css - Reset CSS moderne
    • bretzel-print.css - Styles d'impression
    • bretzel-layouts.css - Utilitaires de mise en page
  2. Layer theme :
    • theme.css - Thème principal
    • theme-tokens.css - Variables CSS du design system
  3. Layer components :
    • styles.css - Styles de base des composants
    • + tous les fichiers CSS des composants (chargés dynamiquement)
  4. Layer utilities :
    • bretzel-utilities.css - Classes utilitaires
