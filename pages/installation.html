Il n'y a volontairement pas de mode d'installation automatisé de KNACSS. Il est vivement conseillé de prendre le temps de découvrir les composants, puis de récupérer les fichiers nécessaires manuellement pour les adapter à vos besoins spécifiques (WordPress, frameworks, etc.).

Téléchargement du fichier de styles

Le fichier natives.css contient l'ensemble des styles pour tous les composants natifs HTML présentés dans ce guide. Ce fichier unique centralise tous les éléments : boutons, formulaires, tableaux, dialogs, etc.

Télécharger natives.css

Pré-requis nécessaires

Pour que les styles de KNACSS fonctionnent correctement, deux éléments sont indispensables :

  • Un reset CSS (reset.alsacreations.com) — Obligatoire
    Le reset permet de partir d'une base neutre en éliminant les styles par défaut des navigateurs.
  • Un thème de couleurs personnalisé — Fortement conseillé
    Définissez vos propres variables CSS de couleurs pour adapter KNACSS à votre charte graphique.

Intégration dans votre projet

Une fois téléchargé, importez le fichier natives.css dans votre projet après le reset CSS et avant vos styles personnalisés : 

<link rel="stylesheet" href="reset.css">
<link rel="stylesheet" href="vos-variables.css">
<link rel="stylesheet" href="natives.css">
<link rel="stylesheet" href="vos-styles.css">

Personnalisation

Chaque composant de KNACSS peut être personnalisé via des variables CSS. Consultez la section "Variables CSS du composant" de chaque élément pour découvrir les options de personnalisation disponibles.
