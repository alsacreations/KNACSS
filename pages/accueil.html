KNACSS apporte des styles modernes et accessibles aux les éléments HTML natifs courants tels que les boutons, champs de formulaires, tableaux, et bien d'autres.

Comment s'en servir ?

Grâce à la barre latérale, chaque composant est présenté de manière interactive avec le code source affichable et copiable. Naviguez dans les différents éléments pour découvrir leurs styles et personnalisations possibles.

En pratique

  • Le fichier natives.css contient tous les styles des composants natifs.
  • Chaque élément peut être modifié à l'aide de variables CSS personnalisables.
  • Les styles sont conçus pour être accessibles et respecter les bonnes pratiques du web.
